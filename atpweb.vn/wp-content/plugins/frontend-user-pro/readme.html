Frontend User Pro

Now create frontend registration, login, guest post, survey forms, event registrations easily even if you are not a programmer!

Frontend User Pro is a WordPress plugin designed to make it easy for you to create frontend registrations, login, guest post, survey forms, event registration and even uploading woocommerce or easy digital downloads product from frontend easily. You can embed forms into any post, page or even widget.

Features

  • Create Unlimited Registration forms.
  • Create Unlimited Forms.
  • Auto Responder, Respond user as they submit post or get register on your site.
  • Conditional Menu, show different menu based on role, or login status.
  • Content Protector, Show content based on role or login status.
  • Get notified for every post user submit on your website.
  • Create custom emails for every approved user.
  • Support Custom Post type.
  • Supports Meta Key.
  • Supports Custom Taxtonomies.
  • Support different Post status.
  • Support Deafult post category.
  • Create different role for different registration forms.
  • Fully supported, professionally written and regularly updated software.

Compatability

Frontend User Pro has been fully tested and working on WordPress version 3.5.2+ including the latest version 4.4. The plugin will be updated in the future if WordPress updates any functionality it relies upon. Frontend User Pro will also work with any theme and you can easily style the forms if you require.

As always, there is full support available at: frontenduser.com
